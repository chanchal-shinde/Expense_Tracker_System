# Expense_Tracker_System

A Python and MySQL based Expense Tracker application that helps users manage daily expenses efficiently.

## Features
- Add expense records
- View expenses in tabular format
- Delete expense records
- Automatic current date assignment
- MySQL database integration
- Menu-driven user interface

## Technologies Used
- Python
- MySQL
- mysql-connector-python

## Concepts Used
- CRUD Operations
- Database Connectivity
- SQL Queries
- Functions
- Loops and Conditional Statements

## Database Table Structure

| Column Name   | Data Type |
|---------------|-----------|
| id            | INT (Primary Key) |
| title         | VARCHAR(100) |
| amount        | FLOAT |
| category      | VARCHAR(50) |
| expense_date  | DATE |
